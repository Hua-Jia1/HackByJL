## 巡检流程

### 巡检路线示意

```mermaid
flowchart TB
    Start[进机房] --> ESD1[戴静电环]
    ESD1 --> Door[进机房门]

    Door --> Row1[机柜排 A 前]
    Row1 --> Inspect1[看设备指示灯]
    Inspect1 --> Sense1[听风扇异响]

    Sense1 --> Row2[机柜排 A 后]
    Row2 --> Line[检查线缆标签]
    Line --> Temp[测温 - 红外]

    Temp --> Row3[机柜排 B]
    Row3 --> PDU[看 PDU 电流]
    PDU --> Spare[看备用资源]

    Spare --> Out[出机房]
    Out --> ESD2[摘静电环]
    ESD2 --> End[登记记录]

    style Start fill:#c8e6c9
    style End fill:#c8e6c9
    style Temp fill:#ffcdd2
    style PDU fill:#fff9c4
```

### 设备前面板灯位示意

```mermaid
flowchart LR
    subgraph Panel["设备前面板"]
        PWR1[PWR 电源灯]
        PWR2[PWR 电源灯 2]
        SYS1[SYS 系统灯]
        FAN[FAN 风扇灯]
        LINK1[LINK 业务灯 1]
        LINK2[LINK 业务灯 2]
    end

    PWR1 -.绿.-> OK1[✅ 正常]
    PWR1 -.灭.-> Bad1[❌ 电源故障]
    PWR1 -.红.-> Bad1

    SYS1 -.闪.-> OK2[✅ 正常]
    SYS1 -.常亮.-> Bad2[❌ 系统故障]
    SYS1 -.灭.-> Bad3[❌ 未启动]

    LINK1 -.绿/闪.-> OK3[✅ 正常]
    LINK1 -.灭.-> Bad4[❌ 链路断]

    style OK1 fill:#c8e6c9
    style OK2 fill:#c8e6c9
    style OK3 fill:#c8e6c9
    style Bad1 fill:#ffcdd2
    style Bad2 fill:#ffcdd2
    style Bad3 fill:#ffcdd2
    style Bad4 fill:#ffcdd2
```
