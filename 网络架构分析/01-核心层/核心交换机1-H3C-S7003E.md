| 项目 | 值 |
|------|-----|

| 4093 | — | 192.168.1.2/24 | 互联管理 |

## 关键接口

| 接口 | 配置 | 说明 |
|------|------|------|
| Bridge-Aggregation1 | trunk vlan 10/20/30/4094 | 与 SW2 互联(Bridge-Agg) |
| GE1/0/1 | trunk vlan 10/20/30/4094,agg 1 | 与 SW2 互联口 1 |
| GE1/0/2 | trunk vlan 10/20/30/4094,agg 1 | 与 SW2 互联口 2 |
| GE1/0/24 | access vlan 2 | →防火墙 F1000 G1/0/0 |
| GE2/0/1 | trunk vlan 10/20/30/4094 | →汇聚交换机 S5120V2 G1/0/5(注:防火墙配置中描述为 SW-2 端口) |
| GE0/0/1 | access vlan 30 | 本地接入(配 packet-filter) |
| GE0/0/2 | access vlan 10 | 本地接入 |
| GE0/0/3 | access vlan 20 | 本地接入 |
| GE0/0/4 | access vlan 30 | 本地接入 |

## VRRP 配置

```
interface Vlan-interface10
 vrrp vrid 1 virtual-ip 192.168.10.254
 vrrp vrid 1 priority 110        ← 主

interface Vlan-interface20
 vrrp vrid 2 virtual-ip 192.168.20.254
 vrrp vrid 2 priority 110        ← 主

interface Vlan-interface30
 vrrp vrid 3 virtual-ip 192.168.30.254
 vrrp vrid 3 priority 100(默认)  ← 备
```

## STP 多实例

```
stp region-configuration
 region-name YKZZY
 instance 1 vlan 10 20
 instance 2 vlan 30
 active region-configuration

stp instance 1 root primary       ← VLAN 10/20 主根
stp instance 2 root secondary     ← VLAN 30 备根
stp global enable
```

## 静态路由

```
ip route-static 0.0.0.0 0 192.168.2.2                # →防火墙 FW
ip route-static 10.119.0.1 32 192.168.1.1            # VPN/远端主机
ip route-static 10.120.147.4 30 192.168.1.1          # VPN 网段
ip route-static 172.31.0.0 16 192.168.30.245         # 远程办公
```

## 安全策略(ACL)

```
acl advanced 3000
 rule 5 deny ip source 192.168.20.0 0.0.0.255 destination 192.168.10.0 0.0.0.255
 rule 10 deny ip source 192.168.30.0 0.0.0.255 destination 192.168.10.0 0.0.0.255
 rule 15 permit ip
# 应用到 Vlan10 入/出方向:禁止医技/科室访问行政办公
```

## 管理与登录

- SSH 启用,ACL 3005 限制源 IP(`192.168.10.111` 单点)
- 日志主机:`192.168.40.16`
- 本地用户:`zkyc`(network-admin + network-operator)
- 登录失败锁定:5 次锁定 10 分钟
