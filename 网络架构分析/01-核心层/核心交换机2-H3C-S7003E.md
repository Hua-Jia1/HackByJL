# 核心交换机 2 — H3C S7003E

> **机柜位置**:U15
> **角色**:H3C 内网核心交换机 2(VRRP 主备切换)
> **原始文件**:`华三核心交换机2.docx`

## 基本信息

| 项目 | 值 |
|------|-----|
| 主机名 | YKZZY-HEXIN-SW-2 |
| 软件版本 | H3C Comware V7.1.070, Release 7557P03 |
| 设备型号 | S7003E |
| 管理 IP | Vlan-interface4094: 192.168.100.9/24 |
| VRRP 状态 | VLAN 10/20 备(默认 100),VLAN 30 主(优先级 110) |
| STP 实例 | 实例 2(VLAN 30)主根,实例 1(VLAN 10/20)备根 |
| 区域名 | YKZZY |

## VLAN 与三层接口

| VLAN | 描述 | 接口 IP | 角色 |
|------|------|---------|------|
| 1 | 默认 | — | 关闭 |
| 3 | to-YKZZY-FW:G1/0/1 | 192.168.3.1/24 | 互联防火墙 |
| 10 | xingzhengbangong | 192.168.10.2/24 | VRRP 备 |
| 20 | yijishebei | 192.168.20.2/24 | VRRP 备 |
| 30 | suoyoukeshi | 192.168.30.2/24 | VRRP 主(优先级 110) |
| 4094 | management | 192.168.100.9/24 | 带外管理 |

## 关键接口

| 接口 | 配置 | 说明 |
|------|------|------|
| Bridge-Aggregation1 | trunk vlan 10/20/30/4094 | 与 SW1 互联 |
| GE1/0/1 | trunk, agg 1 | 与 SW1 互联口 1 |
| GE1/0/2 | trunk, agg 1 | 与 SW1 互联口 2 |
| GE1/0/24 | access vlan 3 | →防火墙 F1000 G1/0/1 |
| GE2/0/1 | trunk vlan 10/20/30/4094 | →汇聚交换机 S5120V2 G1/0/6 |

## VRRP 配置

```
interface Vlan-interface10
 vrrp vrid 1 virtual-ip 192.168.10.254
 vrrp vrid 1 priority 100(默认)  ← 备

interface Vlan-interface20
 vrrp vrid 2 virtual-ip 192.168.20.254
 vrrp vrid 2 priority 100(默认)  ← 备

interface Vlan-interface30
 vrrp vrid 3 virtual-ip 192.168.30.254
 vrrp vrid 3 priority 110        ← 主
```

## STP 多实例

```
stp region-configuration
 region-name YKZZY
 instance 1 vlan 10 20
 instance 2 vlan 30
 active region-configuration

stp instance 1 root secondary   ← VLAN 10/20 备根
stp instance 2 root primary     ← VLAN 30 主根
stp global enable
```

## 静态路由

```
ip route-static 0.0.0.0 0 192.168.3.2           # →防火墙 FW
ip route-static 172.31.0.0 16 192.168.30.245    # 远程办公
```

> 与 SW1 不同:SW2 没有 `10.119.0.1` 和 `10.120.147.4/30` 的静态路由,可能因为这些网段仅需通过 SW1 出。

## 安全策略

- ACL 3000 同样应用在 Vlan10 接口,禁止 VLAN 20/30 访问 VLAN 10
- ACL 3005 限制 SSH 源 IP
- 日志主机 `192.168.40.16`

## 与 SW1 的对照

| 项目 | SW1 (HEXIN-SW-1) | SW2 (HEXIN-SW-2) |
|------|-------------------|-------------------|
| 防火墙互联 VLAN | VLAN 2 (192.168.2.1) | VLAN 3 (192.168.3.1) |
| 防火墙 G1/0/x | G1/0/0 | G1/0/1 |
| 汇聚互联 GE2/0/1 | →SW G1/0/5 | →SW G1/0/6 |
| VRRP VLAN 10/20 | 主 | 备 |
| VRRP VLAN 30 | 备 | 主 |
| STP 实例 1 主 | SW1 | — |
| STP 实例 2 主 | — | SW2 |
