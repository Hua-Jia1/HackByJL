## 安全策略(security-policy ip)

| Rule# | 名称 | 源 → 目的 | 动作 | 备注 |
|-------|------|-----------|------|------|
| 0 | trust-dmz | Trust → DMZ | pass | Client 对象组 → 服务器,服务 C-S |
| 1 | dmz-client | DMZ → Trust | pass | 服务器 → Client,无服务限制 |
| 2 | local-dmz | Local → DMZ | pass | 防火墙自身到 DMZ |
| 3 | trust-local | Trust → Local | pass | 客户端到防火墙 |
| 4 | local-trust | Local → Trust | pass | 防火墙到客户端 |
| 5 | dmz-local | DMZ → Local | pass | DMZ 到防火墙 |
| 7 | untrust_local | Untrust → Local | pass | 外部到防火墙 |
| 8 | local_untrust | Local → Untrust | pass | 防火墙到外部 |
| 9 | dmz_untrust | DMZ → Untrust | pass | 服务器到协同门诊 |
| 10 | untrust_dmz | Untrust → DMZ | pass | 协同门诊到服务器(服务 C-S) |
| 11 | trust_untrust | Trust → Untrust | pass | 内网用户出互联网 |

> 区域间默认拒绝(没有显式 rule 即 drop),只有以上 11 条显式放行。

## 攻击防御(attack-defense policy aaa)

- 扫描防御:中级(drop + log)
- Flood 防御:Syn/Ack/Syn-Ack/Rst/Fin/UDP/ICMP/ICMPv6/DNS/HTTP 全开 + client-verify
- 签名检测:fragment, impossible, teardrop, tiny-fragment, ip-option-abnormal, smurf, traceroute, ping-of-death, large-icmp, large-icmpv6, tcp-invalid-flags, tcp-null-flag, tcp-all-flags, tcp-syn-fin, tcp-fin-only, land, winnuke, udp-bomb, snork, fraggle, icmp-type 255
