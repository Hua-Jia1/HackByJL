## CAPWAP 与 AC 注册

```
wlan-config 1 Zangyiyang
 ssid-code utf-8
 tunnel local

ap-group default
 interface-mapping 1 1 ap-wlan-id 1

ac-controller
 capwap ctrl-ip 3.3.33.3    ← 使用 Loopback 0 作为 CAPWAP 源
 no ac-control disable       ← 启用 AC 控制
```

> 本机不仅是网关,还充当轻量 AP 控制器角色(SSID 1)。AP 通过 W-core DHCP option 138 找到 AC,但本机也配置了 CAPWAP。

```
wlansec 1

 security rsn akm psk enable
 security rsn akm psk set-key ascii ruijie@123

 security wpa akm psk enable
```

- PSK 密码:**ruijie@123**(明文,需加固)

## DHCP

### pool_Gi0/0
```
lease 0 8 0
network 192.168.2.0 255.255.255.0
dns-server 114.114.114.114
default-router 192.168.2.1
```

### pool_Gi0/6
```
lease 0 8 0
network 192.168.255.128 255.255.255.128
dns-server 114.114.114.114
default-router 192.168.255.254
```

> 与 W-core 同段 DHCP。W-core 也有自己的 VLAN 255 SVI,本机 Gi0/6 也提供同段 DHCP,实际看谁是主。

## NAT

```
ip nat pool nat_pool prefix-length 24
 address interface GigabitEthernet 0/7 match interface GigabitEthernet 0/7
ip nat inside source list 1 pool nat_pool overload
```

- NAT 出接口:Gi0/7(WAN)
- ACL 1:permit any

## 路由

```
ip route 0.0.0.0 0.0.0.0 GigabitEthernet 0/7 10.2.2.1      # 默认 → ISP
ip route 10.30.30.0 255.255.255.0 GigabitEthernet 0/6 192.168.255.253  # →W-core VLAN 1022
ip route 192.168.0.0 255.255.0.0 GigabitEthernet 0/6 192.168.255.253   # →W-core 全网段
```
