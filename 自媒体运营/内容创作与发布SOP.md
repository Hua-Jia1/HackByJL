## 一、图文创作 SOP（公众号/知乎/小红书）

### 1.1 文章结构模板

```
【标准图文结构】
┌─────────────────────────────┐
│  标题（按选题SOP的标题公式）   │  ← 决定 70% 打开率
├─────────────────────────────┤
│  开场钩子（1-3句话）          │  ← 决定是否继续读
│  - 痛点提问 / 反常识 / 故事  │
├─────────────────────────────┤
│  正文（三段式）               │
│  段1：是什么（问题/现象）     │
│  段2：为什么（原因/分析）     │
│  段3：怎么办（解决方案）      │
├─────────────────────────────┤
│  金句/总结                    │  ← 记忆点
├─────────────────────────────┤
│  互动引导 + CTA               │  ← 点赞/收藏/评论/关注
└─────────────────────────────┘
```

### 1.2 开头钩子的 5 种写法

| 类型 | 写法 | 示例 |
|------|------|------|
| 痛点共鸣 | 说出用户正在经历的痛苦 | "投了100份简历，一个面试都没有，是不是你？" |
| 反常识 | 挑战普遍认知 | "其实面试官最不看重的就是你的GPA" |
| 故事开场 | 短故事制造代入感 | "昨天一个学员哭着给我打电话，说她被裁了..." |
| 数据冲击 | 用数据制造紧迫感 | "2026年应届生求职人数突破1200万" |
| 结果承诺 | 直接说能获得什么 | "读完这篇，你就能写出让HR停不下来的简历" |

### 1.3 正文创作规范

**公众号长文规范**：
- 段落长度：手机端不超过5行（约150字），超过必分段
- 配图密度：每300-500字配1张图，避免文字墙
- 重点加粗：每篇关键结论/金句加粗，方便扫读
- 小标题：每段前用 ### 小标题分隔，结构清晰
- 字数：干货文 1500-3000字，故事文 800-2000字

**小红书图文规范**：
- 字数：300-800字（单篇）
- 每段不超过3行，多用换行和emoji分隔
- 关键词密度：标题含2-3个领域关键词
- 话题标签：文末 5-10 个 #话题
- 体感：轻松、口语化，像在跟朋友聊天

**知乎回答规范**：
- 字数：高质量回答建议 1500-5000字
- 开头：前50字定生死（会被折叠前展示）
- 结构：总-分-总，引用数据/论文增强可信度
- 排版：知乎编辑器支持 Markdown，善用标题层级

### 1.4 结尾 CTA 模板

```
【互动引导】
- 求赞求收藏："觉得有用就点个赞+收藏，下次找得到"
- 评论引导："你遇到过类似情况吗？评论区聊聊"
- 关注引导："想看更多XX干货，关注我每周更新"
- 私域引流："需要【XX资料包】的，评论区扣1我发你"
```

## 二、短视频创作 SOP（抖音/小红书/B站）

### 2.1 短视频黄金结构

```
【30-60秒短视频脚本结构】
0-3秒   → 钩子画面 + 钩子文案（"你是不是..." / "千万别..." / "教你一招..."）
3-8秒   → 痛点放大（"80%的人都犯过这个错"）
8-45秒  → 干货输出（3点法：第一...第二...第三...）
45-55秒 → 总结金句
55-60秒 → 引导互动 + 关注
```

### 2.2 B站中长视频结构

```
【5-15分钟中长视频结构】
0-30秒    → 片头钩子（本期精彩片段预览 or 提问）
30秒-2分钟 → 问题引入（为什么要讲这个？你的痛点是什么？）
2-10分钟   → 核心内容（分3-5个段落，每段有小标题）
10-13分钟  → 总结回顾（帮用户划重点）
13-结束    → 下期预告 + 三连引导
```

### 2.3 拍摄前检查清单

- [ ] 脚本已完成（至少到分镜头级别）
- [ ] 拍摄设备准备（手机够用，相机更好）
- [ ] 灯光 OK（自然光或补光灯，面光均匀）
- [ ] 收音 OK（安静环境 or 领夹麦）
- [ ] 背景干净/有调性（不在杂乱环境拍）
- [ ] 服装/妆容符合人设
- [ ] B-roll 素材已准备（产品/截图/场景切换素材）

### 2.4 剪辑规范

| 平台 | 画幅 | 时长 | 字幕 | 节奏 |
|------|------|------|------|------|
| 抖音 | 9:16 竖屏 | 15-60s | 必加字幕 | 快节奏，2-3秒一切镜 |
| 小红书 | 9:16 竖屏 / 3:4 | 30-90s | 必加字幕 | 中快，画面精致 |
| B站 | 16:9 横屏 | 5-15min | 建议加字幕 | 按内容节奏，可慢可快 |
| 视频号 | 9:16 竖屏 / 16:9 | 30-60s / 1-5min | 必加字幕 | 中速，适合中老年 |

## 三、封面/首图制作规范

### 3.1 各平台封面尺寸

| 平台 | 尺寸 | 格式 | 大小限制 |
|------|------|------|---------|
| 公众号首图 | 900×383（2.35:1） | JPG/PNG | ≤10MB |
| 公众号次图 | 200×200（1:1） | JPG/PNG | ≤10MB |
| 抖音封面 | 1080×1920（9:16） | JPG | 美观即可 |
| 小红书封面 | 1080×1440（3:4） | JPG/PNG | ≤20MB |
| B站封面 | 1146×717（16:10） | JPG/PNG | ≤5MB |

### 3.2 封面设计原则

1. **大字标题**：封面文字要能在手机缩略图中看清（至少覆盖 30% 面积）
2. **色彩对比**：文字与背景高对比度，不用浅色字+浅色底
3. **人物+文字**：有人脸的封面点击率高 30%+（可选项）
4. **统一模板**：同一系列用统一配色/版式，强化辨识度
5. **小红书特有**：封面字要大且美，要有"杂志感"

## 四、内容审核清单

发布前逐项检查：

### 合规审核
- [ ] 无违禁词/敏感词（可用零克查词/句易网检测）
- [ ] 不涉及时政敏感话题
- [ ] 数据引用有来源标注
- [ ] 图片/视频素材无版权问题（不使用未授权素材）
- [ ] 商业内容标注"广告"/"合作"（合规要求）

### 内容质量审核
- [ ] 标题是否有吸引力（是否用了标题公式）
- [ ] 开头前 3 秒/前 50 字是否够抓人
- [ ] 正文有逻辑、有干货、不水
- [ ] 无错别字/语病
- [ ] 配图/视频清晰度 OK
- [ ] CTA 引导完成了

### 人设一致性审核
- [ ] 语气/调性符合人设（不会突然换风格）
- [ ] 内容与账号定位一致（不跑偏）

## 五、发布 SOP

### 5.1 发布前准备（发布前 2 小时）

- [ ] 终稿已审核通过
- [ ] 封面图/视频封面已就绪
- [ ] 各平台素材已适配完毕（尺寸/格式）
- [ ] 话题标签已准备好（5-10个）
- [ ] 评论区预埋评论已写好（2-3条引导互动的评论）

### 5.2 最佳发布时间

| 平台 | 工作日最佳 | 周末最佳 | 说明 |
|------|----------|---------|------|
| 公众号 | 早7:30 / 晚20:30 | 早8:00 | 通勤+睡前 |
| 抖音 | 午12:00 / 晚18:00-21:00 | 同工作日 | 午休+下班后 |
| 小红书 | 早7:30 / 午12:00 / 晚19:00-22:00 | 早9:00 / 晚20:00 | 三个黄金时段 |
| B站 | 午12:00 / 晚18:00 | 午12:00 | 午休+晚饭后 |
| 知乎 | 早8:00-10:00 / 晚20:00-22:00 | 同工作日 | 摸鱼时间+睡前 |
| 视频号 | 午12:00 / 晚18:00-20:00 | 午12:00 | 朋友圈活跃时段 |

> **规则**：固定发布时间，培养用户期待感。宁可推迟不可提前（提前打乱用户习惯）。

### 5.3 发布后 1 小时动作

- [ ] 用自己的号点赞/收藏/评论（启动冷启动）
- [ ] 回复前 10 条评论（越快越好，影响算法推荐）
- [ ] 转发到朋友圈/社群（私域助推）
- [ ] 各平台检查发布是否正常（封面/排版/链接无 bug）

### 5.4 发布后 24 小时动作

- [ ] 记录初始数据（发布 1h / 3h / 24h 数据）
- [ ] 回复剩余评论
- [ ] 观察数据表现，判断是否需要投流助推
