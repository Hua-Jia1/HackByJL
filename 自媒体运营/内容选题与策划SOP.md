# 内容选题与策划 SOP

> **所属模块**: 自媒体运营 — 阶段二  
> **前置条件**: [账号定位与规划 SOP](./账号定位与规划SOP.md) 已完成  
> **产出物**: 选题库、周内容日历、标题库  
> **更新频率**: 选题库每周补充；内容日历每周日前完成下周排期

## 一、选题来源体系

### 1.1 八大选题来源

| 序号 | 来源 | 具体方法 | 工具/渠道 |
|------|------|---------|----------|
| 1 | **对标账号** | 拆解竞品爆款选题，做"同题异构" | 新榜、飞瓜、手动刷 |
| 2 | **评论区挖掘** | 从自己和竞品评论区找用户真实问题 | 各平台评论区 |
| 3 | **搜索下拉词** | 输入领域关键词，看搜索联想/相关搜索 | 各平台搜索框 |
| 4 | **热点日历** | 节假日、行业节点、社会热点 | 营销日历、微博热搜 |
| 5 | **跨界迁移** | 其他领域的爆款逻辑迁移到你的赛道 | 不同赛道的首页推荐 |
| 6 | **数据工具** | 用数据工具找上升趋势话题 | 5118、巨量算数 |
| 7 | **个人经历** | 自己的成功/失败经验、行业内幕 | 个人知识库 |
| 8 | **书籍/课程** | 读书笔记、课程笔记的二次创作 | 微信读书、得到 |

### 1.2 选题优先级：常青 > 热点

| 类型 | 定义 | 生命周期 | 建议占比 |
|------|------|---------|---------|
| 常青选题 | 不随时间衰减的刚需内容 | 6个月以上 | 60% |
| 周期选题 | 每年固定时间会火的内容 | 1-2个月 | 25% |
| 热点选题 | 48小时内有流量红利的突发内容 | 1-7天 | 15% |

> **原则**: 常青选题做基本盘，热点选题做增长爆点。不要为了追热点打乱内容节奏。

## 二、选题评估标准（选题打分卡）

每个选题入库前，按以下维度打分，总分 ≥ 12 分才进入排期：

| 评估维度 | 权重 | 打分标准(1-5) | 得分 |
|---------|------|-------------|------|
| 搜索需求 | 25% | 5=大需求 3=中等 1=小众 | |
| 竞争程度（越低越高分） | 20% | 5=蓝海 3=一般 1=红海 | |
| 与账号定位匹配度 | 25% | 5=完美匹配 3=相关 1=勉强 | |
| 可操作性 | 15% | 5=能独立完成 3=需协作 1=难落地 | |
| 变现潜力 | 15% | 5=能直接转化 3=间接 1=纯流量 | |
| **总分** | | | **/25** |

> 及格线 12 分，优秀线 18 分。

## 三、选题库管理

### 3.1 选题库结构

使用飞书多维表格 / Notion 建立选题库，包含以下字段：

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| 选题标题 | 文本 | 最终发布的标题 | "面试被问缺点，这样回答直接加分" |
| 内容类型 | 单选 | 图文/短视频/长视频/直播/图文+视频 | 短视频 |
| 选题来源 | 单选 | 见1.1八大来源 | 评论区挖掘 |
| 目标平台 | 多选 | 发布平台 | 小红书、抖音 |
| 选题评分 | 数字 | 按评分卡打分 | 16 |
| 状态 | 单选 | 待评估→已入库→已排期→已发布→已复盘 | 已排期 |
| 发布日期 | 日期 | | 2026-06-05 |
| 互动数据 | 文本 | 发布后回填 | 赞1200/藏800/评150 |
| 复盘笔记 | 文本 | 表现好/差的原因 | "标题用反问句效果好" |

### 3.2 选题库维护节奏

- **每周一**: 从八大来源收集 10-15 个新选题
- **每周二**: 选题评估会（团队）或自评（个人），选出下周 5-7 个选题
- **每周三**: 完成内容日历排期
- **每日**: 刷平台热搜/对标账号，有热点立即入库加急

## 五、标题方法论

### 5.1 爆款标题的 8 种公式

| 公式 | 结构 | 示例 |
|------|------|------|
| 数字清单 | "X个/种/条 + 主题" | "35岁前一定要懂的5条职场潜规则" |
| 痛点+解决方案 | "痛点？试试这个" | "面试总是挂？试试这个回答框架" |
| 反常识/反差 | "你以为...其实..." | "越努力越穷？这3种努力正在毁掉你" |
| 身份标签 | "给[人群]的建议" | "给25岁产品经理的3个忠告" |
| 结果承诺 | "做到X，就能Y" | "掌握这3个技巧，粉丝量翻倍" |
| 紧迫感/稀缺 | "限时/最后/错过就" | "6月求职黄金期，再拖就错过了" |
| 提问唤醒 | "你知道吗/为什么" | "为什么你的简历总是石沉大海？" |
| 故事钩子 | "从...到..." | "从月薪5K到年薪50万，我只做对了一件事" |

### 5.2 标题优化流程

1. **写 5 个备选标题** — 用不同公式各写一个
2. **自检三问**：
   - 用户看到会想点吗？（好奇心）
   - 用户看完会觉得有收获吗？（价值感）
   - 标题和内容一致吗？（不标题党）
3. **A/B 测试**（公众号可用）：首图/标题双版本测点击率
4. **复盘记录**：高点击标题存入"标题素材库"

### 5.3 标题禁忌

- ❌ 纯标题党（打开后内容不匹配 → 高跳出率 → 被限流）
- ❌ 过度使用感叹号和问号
- ❌ 负面情绪绑架（焦虑营销适度）
- ❌ 蹭敏感话题/时政热点（封号风险）

## 六、热点追踪 SOP

### 当出现热点时（15分钟内）

1. **判断是否可追**：
   - [ ] 与赛道相关？（无关不追）
   - [ ] 有没有独特的切入角度？
   - [ ] 能不能在 4 小时内出内容？
   - [ ] 是否有合规风险？

2. **快速产出**：
   - 热点内容时效性第一，质量可放宽到常规的 70%
   - 标题必须含热点关键词
   - 评论第一波抢占前排

3. **热点工具**：
   - 微博热搜 / 抖音热榜 / 百度热搜
   - 新榜热点日历（可预报行业节点）
   - 各平台搜索框"大家都在搜"
