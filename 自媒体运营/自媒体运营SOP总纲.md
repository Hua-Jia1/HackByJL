# 自媒体运营 SOP 总纲

> **版本**: v1.0  
> **适用平台**: 微信公众号 / 抖音 / 小红书 / B站 / 知乎 / 微博 / 视频号  
> **适用团队规模**: 个人创作者 — 5人小团队  
> **文档用途**: 标准化自媒体运营全流程，降低沟通成本，提升内容产出效率与质量

---

## 一、核心流程全景图

```
账号定位 → 内容选题 → 内容创作 → 审核修改 → 发布分发 → 数据复盘 → 优化迭代
   ↑                                                              ↓
   └──────────────────── 粉丝互动 & 变现闭环 ─────────────────────┘
```

| 阶段 | 核心产出 | 负责人 | 周期 |
|------|---------|--------|------|
| 账号定位 | 定位文档、人设卡片 | 运营负责人 | 一次性（按季度复审） |
| 内容选题 | 选题库、内容日历 | 内容策划 | 每周更新 |
| 内容创作 | 文案/视频/图文初稿 | 内容创作 | 按排期 |
| 审核修改 | 终稿 | 审核人 | 发布前24h |
| 发布分发 | 已发布内容 + 数据埋点 | 运营执行 | 按排期 |
| 数据复盘 | 数据报告、优化建议 | 运营分析 | 每周/每月 |
| 粉丝互动 | 回复记录、粉丝画像更新 | 运营执行 | 每日 |

## 三、团队角色与职责

### 个人创作者模式
- 一人承担全部角色，按SOP流程分时段执行

### 小团队模式（3-5人）

| 角色 | 职责 | 对接文档 |
|------|------|---------|
| 运营负责人 | 账号定位、整体策略、商务对接 | 总纲、变现SOP |
| 内容策划 | 选题挖掘、内容日历、热点追踪 | 选题SOP |
| 内容创作 | 文案撰写、视频拍摄/剪辑、图文制作 | 创作SOP |
| 运营执行 | 排版发布、评论区互动、数据采集 | 发布SOP、数据SOP |
| 审核人（可兼任） | 内容审核、合规检查 | 创作SOP（审核清单） |

## 五、通用工作原则

1. **先模仿再创新** — 新人/新号前3个月以模仿对标账号为主，跑通流程后再加入个人风格
2. **内容为王，发布时间是放大器** — 不要为了赶排期牺牲内容质量
3. **数据驱动决策** — 所有优化必须有数据支撑，不做"我觉得"
4. **一次创作，多平台分发** — 同一选题适配各平台后分发，最大化内容利用率
5. **合规底线** — 广告法、平台规则、版权问题零容忍
6. **持续迭代** — 每周复盘，每月迭代SOP本身
