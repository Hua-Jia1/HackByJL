# 账号定位与规划 SOP

> **所属模块**: 自媒体运营 — 阶段一  
> **前置条件**: 无  
> **产出物**: 账号定位文档、人设卡片、竞品分析表  
> **复审周期**: 每季度一次

---

## 一、定位四步法

### Step 1: 明确变现模式（以终为始）

在开始做内容前，先确定赚钱路径。常见自媒体变现模式：

| 变现模式 | 适合领域 | 门槛 | 收入天花板 |
|---------|---------|------|-----------|
| 流量主 / 平台分成 | 泛娱乐、资讯、情感 | 低（粉丝量门槛） | 中低 |
| 知识付费 | 职场、教育、理财、情感咨询 | 中（需专业度） | 高 |
| 品牌商单 / 广告 | 美妆、数码、生活方式、知识 | 中（需粉丝基础） | 中高 |
| 直播带货 | 全品类 | 中（需供应链/话术） | 极高 |
| 私域引流 | 教育、保险、房产、B2B | 高（需转化能力） | 高 |
| IP授权/出版 | 漫画、知识、文学 | 高 | 不定 |

**执行动作**：
- [ ] 选定 1-2 个变现模式，写入定位文档
- [ ] 调研该模式下头部账号的收入结构（飞瓜/星图可查）

---

### Step 2: 选定垂直赛道

垂直 = 领域 × 人群 × 内容形态 的交集。

**赛道选择矩阵**：

| 评估维度 | 权重 | 打分(1-5) | 加权分 |
|---------|------|----------|--------|
| 个人兴趣/擅长程度 | 30% | | |
| 市场规模与增速 | 20% | | |
| 竞争激烈程度（越低越好） | 20% | | |
| 变现效率 | 20% | | |
| 内容可持续性 | 10% | | |

> **原则**: 兴趣 + 市场 + 竞争度三者交集为最佳赛道。宁可做小池子里的大鱼。

**常见垂直赛道举例**：
- 职场：简历优化 / 面试技巧 / 职场沟通 / 大厂生存
- 理财：基金定投 / 保险配置 / 存钱技巧 / 副业搞钱
- 生活：极简生活 / 收纳整理 / 早起打卡 / 独居vlog
- 知识：读书笔记 / 历史解读 / 语言学习 / 编程入门

**执行动作**：
- [ ] 列出 3-5 个候选赛道，完成打分矩阵
- [ ] 选定 1 个主赛道 + 1 个备选赛道
- [ ] 在选定赛道中找出 10 个对标账号

---

### Step 4: 提炼核心差异点（USP）

回答三个问题：
1. **用户为什么看你而不是看别人？**
2. **你能提供什么独特价值？**（信息差、认知差、情绪价值、陪伴感）
3. **用一句话介绍你的账号**（即"电梯演讲"）

公式：`我为[目标人群]提供[独特价值]，帮助他们[解决什么痛点]，不同于[竞争对手]，我[差异化特点]。`

**示例**：
- "我为25-35岁职场女性提供实用的极简穿搭方案，帮助她们用最少的衣服穿出质感，不同于传统穿搭博主只堆品牌，我专注于一衣多穿和基础款搭配。"

**执行动作**：
- [ ] 完成电梯演讲稿
- [ ] 找 3 个目标用户测试，看他们能否 10 秒内理解你在做什么

## 二、人设打造指南

### 人设 = 专业标签 + 性格特质 + 视觉符号

| 人设要素 | 说明 | 示例 |
|---------|------|------|
| 专业标签 | 你的核心能力标签（1-2个词） | "前大厂HR" / "野生设计师" |
| 性格特质 | 用户记住的性格印象 | 毒舌但有用 / 温柔治愈 / 干货强迫症 |
| 口头禅 | 固定出现的记忆点语句 | "懂的都懂" / "说点大实话" |
| 视觉符号 | 每次出现必有的视觉元素 | 固定开场动作 / 特定色调 / 标志性道具 |

**执行动作**：
- [ ] 完成人设卡片（含上述四要素）
- [ ] 检查人设是否"真实可信"——过度包装的人设迟早崩塌

## 三、竞品分析 SOP

### 3.1 找对标账号

三种对标层级：

| 层级 | 定义 | 数量建议 | 用途 |
|------|------|---------|------|
| 榜样级 | 赛道头部，你最终想成为的样子 | 1-2个 | 方向参考 |
| 学习级 | 比你高1-2个台阶，可模仿可追赶 | 3-5个 | 拆解学习 |
| 警戒级 | 同量级但起号快，直接竞品 | 3-5个 | 差异化分析 |

## 四、账号装修规范

### 4.1 各平台必填项

| 项目 | 公众号 | 抖音 | 小红书 | B站 | 知乎 |
|------|--------|------|--------|-----|------|
| 头像 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 昵称 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 简介 | ✓ | ✓ | ✓ | ✓ | ✓ |
| 背景图 | - | ✓ | - | ✓ | - |
| 封面模板 | 首图 | 封面 | 封面 | 封面 | 头图 |

### 4.2 命名规则

- **好名字三原则**：易记（不超过5个字）、易搜（避免生僻字）、易关联（包含领域关键词）
- **推荐公式**：`[领域/身份] + [昵称/特点]`
  - 例：`设计师阿杰`、`北京买房指南`、`老赵聊职场`
- **避坑**：不用火星文、emoji堆叠、频繁改名（半年内不宜改名）

### 4.3 简介撰写公式

**第一行**：我是谁 + 专业背书  
**第二行**：我能给你什么价值  
**第三行**：引导关注 / 行动指令

**示例**：
```
前阿里P7产品经理 | 服务过100+求职者
每天分享互联网面试干货和职场真相
关注我，拿offer不走弯路 👇
```

## 五、定位自检清单

在正式投入内容创作前，逐项检查：

- [ ] 变现模式已明确，且有真实案例验证可行性
- [ ] 主赛道已选定，且至少 10 个对标账号在该赛道存活 6 个月以上
- [ ] 用户画像已完成，且通过真实用户反馈验证
- [ ] 核心差异点在 10 秒内能被目标用户理解
- [ ] 人设四要素（标签/性格/口头禅/符号）已确定
- [ ] 竞品拆解完成至少 10 份，提炼出 5 条可复用爆款因子
- [ ] 账号装修（头像/昵称/简介/封面）已完成，各平台统一
- [ ] 已完成 3 位目标用户 1v1 访谈，确认定位能打动他们
